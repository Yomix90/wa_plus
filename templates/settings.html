Liaison du Webhook OpenWA

Pour que WaPlus reçoive les messages entrants reçus par votre numéro OpenWA, renseignez l'URL suivante dans la configuration des Webhooks de votre instance OpenWA :

https://<votre-domaine-waplus>/webhook

Dans la configuration d'OpenWA, assurez-vous d'abonner cette URL à l'événement message.received afin d'activer les réponses automatiques Gemini.
